Hello Hugo

Jun 1, 2019
1 min read

1、Hugo Install

brew install hugo
# check it
hugo version

2、Create a Blog

hugo new site [path]

3、Create a article

# cd [path],create a article
hugo new post/hello-hugo.md

4、Set Theme

Reference Hugo Website , choose a theme.

# cd [path],and clone it
git clone https://github.com......

Set up config.toml file, for example:

baseURL = "http://example.org/"
languageCode = "en-us"
title = "Htzhu's blog"
# your theme's name
theme = "m10c"

5、Start it

hugo server -D
